Working With Templates

Import Template

One feature of PlasmaGraph is that it has the capability to save all the options chosen for a given graph chart so the user can apply a common set of plotting options to different sets of data. This is achieved by creating a special "template" file that the system uses to pre-set the options in the tool bar used to create graphs.

To use an existing template go to the main menu bar and choose Template >> import. Then select the template file that you want to use.

A dialog box will open asking you to choose a template to import.
