Tabbed Menu (Data)

This menu is used to tell PlasmaGraph which set of values to plot and how these values are presented in the Graph window. You can take a look at the parts of the Graph window here

The menu is divided into 4 sections and the Graph button which produces the graph:

Chart settings

This section lets you set the name and type of the chart. The type can only be "XY Graph" and "Bar Graph".

X Axis Settings

Here you can choose which set of values are assigned to the X axis and what is the text to be displayed in its label. Also, you can choose the scale of the axis which can be numerical or logarithmic

Y Axis Settings

Here you can choose which set of values are assigned to the Y axis and what is the text to be displayed in its label. Also, you can choose the scale of the axis which can be numerical or logarithmic

Grouping Settings

This option lets you group the selected set of values that are going to be graphed by any other set of values stored in the data file.
