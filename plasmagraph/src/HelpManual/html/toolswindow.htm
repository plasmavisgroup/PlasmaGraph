Tools Window

This component is made of two sections, the main menu bar and the tabbed menu bar

Main menu

The main menu lets you perform the following actions:

  • Import data. Click for details
  • Reset data. Click for details
  • View data. Click for details
  • Save template. Click for details
  • Import template. Click for details
  • Display this guide.

Tabbed menu

This menu is used to specify how the imported data is going to be graphed. It is composed of two tabs "Data view" & "Options view"

Note: You can save all options made using these tabs by creating a "template". Click for details

Data view

Here you can choose the graph's title, type, X and Y axis data and whether to group the values by any given column. Click for details

Options view

This tab lets you choose an interpolation type and whether to scan for outliers or not. Click for details
