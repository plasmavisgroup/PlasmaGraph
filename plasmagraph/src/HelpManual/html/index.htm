PlasmaGraph User's Guide

PlasmaGraph is a useful tool for making graph charts from data stored in either a CSV file (*.csv) or a Level 5 MAT-File (*.mat).

Getting started

To plot a set of values against another you have to follow these steps:

  • Upload a file with at least two sets of data. More info.
  • Choose chart title and type. More info.
  • Choose which set of values (column) is going to be used for the "X" axis of the graph. More info.
  • Choose which set of values (column) is going to be used for the "Y" axis of the graph. More info.
  • (Optional) Choose a set of values or column to use for grouping. More info.
  • (Optional) Choose to look for outliers. More info.
  • (Optional) Choose interpolation type. More info.
  • Click the "Graph" button located at the bottom of the Tabbed menu (Data). More info.

Exporting The Graph

To save a graph you just have to right click on it. Two options will come up, one is to copy the image in the system's clipboard and the other one is to export the chart as an image with *.png extension.

Working With Templates

One feature of PlasmaGraph is that it has the capability to save all the options chosen for a given graph chart so the user can apply a common set of plotting options to different sets of data. This is achieved by creating a special "template" file that the system uses to pre-set the options in the tool bar used to create graphs.

To save a template file go to the main menu bar and choose Template >> save template

A dialog box will open asking you to choose where to save the template.

To use an existing template go to the main menu bar and choose Template >> import. Then select the template file that you want to use.

A dialog box will open asking you to choose a template to import.
